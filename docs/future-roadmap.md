# Future Roadmap

## Planned Modules

- Helpdesk / ticketing
- Document repository and versioning
- Rich notifications
- Activity feed UI
- Reporting dashboards and KPI widgets
- REST API endpoints and external connectors

## Near-Term Enhancements

- Replace plain admin tables with `WP_List_Table` implementations.
- Add attachment mappings.
- Add task dependencies and richer subtask hierarchy if needed.
- Add dashboard widgets and richer filters.
- Add front-end member dashboards powered by the existing services.
- Expand notifications beyond mentions to assignment, status change, due-soon, and overdue events.
- Add Kanban board interactions powered by database-backed task statuses.
